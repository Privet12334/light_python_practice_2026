ИНСТРУКЦИЯ ПО ЗАПУСКУ БАЗОВОГО ИНДЕКСАТОРА ПАПОК

1. Убедитесь, что Python 3 установлен.
2. Откройте терминал (командную строку) в папке проекта.
3. Запустите утилиту командой:
   python -m src.main ПУТЬ_К_ПАПКЕ

   Например: python -m src.main C:\Users\User\Documents

4. Дополнительные параметры:
   -e РАСШИРЕНИЕ   – фильтр по расширению (например, -e txt)
   -o ИМЯ_ФАЙЛА    – сохранить отчёт в указанный файл (по умолчанию report.txt)
   -v               – показать дерево папок и файлов
   -h               – показать полную справку

5. После запуска отчёт появится в консоли и будет сохранён в файл (по умолчанию report.txt).

Примеры:
   python -m src.main ~/Documents -v
   python -m src.main ~/Documents -e py -o python_report.txt
